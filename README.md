# Eduardo Almeida – Personal Portfolio Website

This repository contains my personal portfolio website, developed to present my academic background, technical skills, and selected projects in a clear, structured, and professional manner.

The website serves as a central reference point for recruiters, academic evaluation, and personal branding, offering a concise overview of my profile and experience.

---

## Overview

This portfolio provides an organised presentation of my profile, including:

- Academic and professional timeline  
- Selected projects with dedicated detail pages  
- Technical and interpersonal skills  
- Personal introduction and contact information  
- Fully responsive layout with light and dark theme support  

The design focuses on clarity, usability, and long-term maintainability.

---

## Purpose

The main objectives of this project are to:

- Demonstrate proficiency in modern front-end development technologies  
- Document academic and professional progression  
- Provide a professional and accessible online presence  
- Ensure scalability and ease of future maintenance  

---

## Technologies Used

- **Vite** – development server and build tool  
- **React** – component-based user interface library  
- **TypeScript** – static typing for improved reliability and scalability  
- **Tailwind CSS** – utility-first styling framework  
- **shadcn/ui** – accessible and reusable UI components  
- **React Router** – client-side routing  
- **TanStack Query** – asynchronous state management
